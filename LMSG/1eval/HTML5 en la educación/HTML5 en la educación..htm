Descripción del material

 HTML5 pretende mostrar las bases para creación de páginas web usando el estándar de la World Wide Web. Cabe destacar que en este sentido HTML5 no difiere en exceso de HTML en sus estructuras básicas.

El material de HTML5 nos enseña desde cero cómo crear una página web, desde una perspectiva muy práctica, obviando aquellos aspectos que aún no han sido cerrados totalmente por el organismo que diseña el estándar y aquellos otros que no funcionan completamente en los navegadores actuales. El material también es muy cuidadoso con las técnicas que han quedado obsoletas o desaconsejadas en HTML. Ninguna de ellas aparece en el material.

No pretende ser una revisión exhaustiva del estándar HTML5, sino una aproximación a las novedades fundamentales, que unidas a las técnicas más clásicas permitirán al lector generar sus propias páginas web con éxito. Todo ello acompañado de una revisión de las técnicas imprescindibles de CSS3 y una introducción al uso de Javascript.

Los materiales recogen una serie de explicaciones sobre los diferentes elementos que constituyen las páginas web y un conjunto de ejercicios intercalados. Además, para conseguir estos objetivos, contamos con un tema central de trabajo: construir un sitio web moderno, compuesto de varias páginas con sus correspondientes enlaces. Para ello exponemos el contenido necesario en catorce entradas o enlaces principales equivalentes a catorce módulos, a saber:

  • Introducción al lenguaje HTML.
  • Aspectos básicos del lenguaje HTML.
  • Formato de texto.
  • Enlaces.
  • Elementos multimedia.
  • Maquetación de una página web.
  • Organización de la información.
  • Publicación de un proyecto web.
  • Hojas de estilo.
  • Diseño web con estilos.
  • Maquetación con estilos.
  • Formularios y Javascript.
  • Técnicas avanzadas.
  • Contenidos extra.

 Al finalizar el estudio del material, tendremos las nociones necesarias para diseñar sitios web basados en las técnicas más actuales de HTML 5 y CSS3, además de conocer algunos aspectos iniciales de Javascript.

Para reforzar el carácter didáctico del material, cada módulo se acompaña de una propuesta de aplicación en el aula para enseñar HTML al alumnado.
